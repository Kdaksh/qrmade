# QrMade
It's A Quick Tool To Make Qr Codes. You Can Make Your Qr Code Within Few Seconds. You Can Choose The Type Of Qr Code. We Provide Five Types Of Qr Codes That Are Free Text, Mobile Number, Email, Contact, Url.
